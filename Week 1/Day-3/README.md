# Day 1 – Google Dork Library

## 📌 Overview
This practical involved creating a categorized Google Dork Library for efficient OSINT investigations. The library contains search queries for documents, research papers, spreadsheets, and government resources.

## 🎯 Objectives
- Build a reusable Google Dork Library.
- Organize search queries into categories.
- Improve search intelligence skills.

## 🛠️ Tools Used
- Google Search
- Google Dorks
- Web Browser
- GitHub

## 📂 Files
- google dork library..md

## 📖 Topics Covered
- PDF Document Searches
- Excel & CSV Searches
- Research Paper Searches
- Government Document Searches
- Search Intelligence

## ✅ Outcome
Created a structured Google Dork Library containing categorized search queries for efficient and ethical OSINT investigations.

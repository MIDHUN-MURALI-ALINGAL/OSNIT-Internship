# Day 5 – Internet Archive Investigation

## 📌 Overview
This practical focused on analyzing historical versions of websites using the Internet Archive (Wayback Machine) to understand how websites evolve over time.

## 🎯 Objectives
- Explore archived websites.
- Compare website changes across different years.
- Analyze improvements in design, navigation, and content.

## 🛠️ Tools Used
- Wayback Machine
- Google Search
- Web Browser
- Screenshot Tool
- GitHub

## 📂 Files
- Day 5.md

## 📖 Topics Covered
- Internet Archive
- Website Evolution
- Historical Website Analysis
- Website Comparison

## ✅ Outcome
Compared historical versions of OpenAI, Microsoft, and Tesla websites and observed changes in design, services, and online presence.

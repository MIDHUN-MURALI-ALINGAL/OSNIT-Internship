# Week 1 – Day 2 
## DNS Investigation & Website Investigation (Wikipedia)

## 📌 Overview
This practical combined two OSINT activities. The first focused on understanding the Domain Name System (DNS), DNS records, URL structure, and website architecture using **Wikipedia** as the target domain. The second involved investigating Wikipedia's publicly available information to understand its purpose, ownership, history, infrastructure, and online presence.

## 🎯 Objectives
- Understand how the Domain Name System (DNS) works.
- Learn common DNS record types and their purposes.
- Analyze website URL structure and architecture.
- Investigate Wikipedia using publicly available information.
- Understand website ownership, purpose, and organizational details.
- Practice ethical Open Source Intelligence (OSINT) techniques.

# Day 13 – Domain Intelligence

## Overview
This practical introduced Domain Intelligence concepts, including WHOIS lookups, registrars, domain structure, and Top-Level Domains (TLDs).

## Learning Objectives
- Understand domain name structure.
- Perform WHOIS lookups.
- Identify registrars and registration details.
- Compare public and privacy-protected WHOIS records.

## Topics Covered
- Domain Name Structure
- WHOIS
- Registrar Information
- gTLDs
- ccTLDs
- WHOIS Privacy

## Tools Used
- ICANN Lookup
- Whois.com
- MXToolbox
- Web Browser
- GitHub

## Files Included
- DAY 13 Domain Intelligence.md

## Key Learning
Developed practical experience in domain intelligence by analyzing WHOIS records and understanding how domain registration information supports OSINT investigations.
